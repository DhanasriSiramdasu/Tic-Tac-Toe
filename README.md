# ❌⭕ Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**. Two players can take turns placing **X** and **O** on a 3×3 grid, with the game automatically detecting the winner and allowing players to start a new game or reset the board.

---

## 🚀 Features

- 🎮 Two-player gameplay
- ❌⭕ Alternating turns between X and O
- 🏆 Automatic winner detection
- 🔒 Prevents overwriting occupied cells
- 🔄 Reset game functionality
- 🆕 Start a new game after a winner is declared
- 📱 Clean and responsive user interface

## 🎯 How to Play

1. The game starts with **Player O**.
2. Players take turns clicking on empty cells.
3. Once a cell is selected, it cannot be changed.
4. The first player to align **three identical symbols** horizontally, vertically, or diagonally wins.
5. A congratulatory message is displayed for the winner.
6. Click **Reset Game** or **New Game** to play again.

---

## ⚙️ Game Logic

- Maintains player turns using a boolean variable.
- Checks all possible winning combinations after every move.
- Disables all cells once a winner is found.
- Allows restarting the game without refreshing the page.
